Ice Age Tree Explorer

Welcome to the Ice Age Tree Explorer! This interactive map is designed to allow you to explore data on North America's paleoecological history. Select a way of expressing the data below: map view for an overview of pollen data, and data view for more detailed information on sites where data has been collected. Then, use the controls on the map to sequence through time, overlay ice sheets, and retrieve information on science and context.

Data: Pollen data was downloaded from the Neotoma Paleoecology Database. Then, it was temporally interpolated to every 500 years.
